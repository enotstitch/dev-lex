# DevLex

Telegram-бот для изучения технического английского языка разработчиками.

## О проекте

DevLex помогает программистам:

- изучать технические термины;
- лучше понимать документацию;
- тренировать написание названий функций и переменных.

## Технологии

- Node.js
- grammY
- JavaScript
